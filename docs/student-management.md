# Student Management

## Constraints

- Max 4 students per parent
- Avatars: lion, elephant, cheetah, monkey, eagle, fish
- Grades: KG1, KG2, P1, P2, P3

## Flows

- Registration creates first student automatically
- Add student from `/student/add`
- Select student at `/student/select`

## Security

- Parent can only access their own student records
