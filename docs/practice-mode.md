# Practice Mode

## Features

- Topic selection: counting, addition, subtraction, multiplication
- 10 questions per session
- 30 second timer per question
- Instant correctness feedback
- Final results summary and rewards

## Server-Side Behavior

- Session created on start
- Answers persisted per question
- Session stats updated once per question
- Completion computes accuracy and stars

## Rewards

Stars are calculated based on accuracy:

- 90%+ → 50 stars
- 80–89% → 40 stars
- 70–79% → 30 stars
- 60–69% → 20 stars
- below 60% → 10 stars

## Known Limits

- No XP or leveling yet (Phase 4)
- No audio narration yet (Phase 2+)
